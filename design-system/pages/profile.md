# 个人中心 (Profile) — 页面设计覆盖

> 继承 `MASTER.md` 全部规范，以下为个人中心特定的设计规则。

## 页面职责

个人设置：提醒偏好、分类管理入口、库存统计、数据导出。

## 布局结构

```
┌──────────────────────────────┐
│  用户信息区                    │
│  (头像 + "我的美妆库" + 产品数)│
├──────────────────────────────┤
│  库存统计卡片行                │
│  (按分类统计数量)              │
├──────────────────────────────┤
│  设置列表 · 提醒设置           │
│  (过期提前提醒 / 推送 / 频率)  │
├──────────────────────────────┤
│  设置列表 · 管理               │
│  (分类管理 / 已过期 / 数据导出) │
├──────────────────────────────┤
│  底部 TabBar                  │
└──────────────────────────────┘
```

## 特定规则

### 用户信息区

- 背景使用 `--color-primary-bg`
- 头像圆形，`--radius-full`，48x48px
- 使用微信头像或默认 SVG 头像
- 主标题 "我的美妆库"（H2），副标题 "共 X 件产品"（Caption）

### 库存统计

- 按分类展示数量，使用紧凑统计卡片
- 每个卡片：数字（`--font-stat` 16px级别）+ 分类名
- 卡片底色使用分类对应的浅色（护肤→粉、彩妆→紫、其他→绿等）

### 设置列表

- 使用分组列表样式，每组有标题（Caption, `--color-text-tertiary`）
- 列表项高度 48px
- 右侧展示当前值 + 箭头图标（SVG chevron-right）
- 分隔线使用 `--color-border`，缩进到文字对齐位置
- 开关控件（推送通知）使用 `--color-primary` 作为激活色
