# 添加页 (Add) — 页面设计覆盖

> 继承 `MASTER.md` 全部规范，以下为添加页特定的设计规则。

## 页面职责

产品录入：淘宝链接粘贴解析 + 手动表单录入，双模式切换。

## 布局结构

```
┌──────────────────────────────┐
│  模式切换 Tab (链接导入 | 手动)│
├──────────────────────────────┤
│  [链接模式]                   │
│  链接输入框 + 解析按钮         │
│  解析结果提示                  │
├──────────────────────────────┤
│  表单区域                     │
│  品牌 / 名称 / 分类 / 规格    │
│  生产日期 / 保质期             │
│  过期时间自动计算展示          │
├──────────────────────────────┤
│  保存按钮                     │
├──────────────────────────────┤
│  底部 TabBar                  │
└──────────────────────────────┘
```

## 特定规则

### 模式切换 Tab

- 使用胶囊样式切换：选中态 `--color-primary` 填充 + 白色文字，未选中 `--color-bg` + `--color-text-secondary`
- 圆角 `--radius-button` (12px)
- 切换动画 `--duration-normal` (200ms)

### 链接输入区

- 输入框 + 按钮横向排列，按钮使用 `--color-primary`
- 解析中状态：按钮显示加载指示器，禁用重复点击
- 解析成功：显示绿色提示卡片（`--color-safe-bg` 底色），展示识别到的产品名
- 解析失败：显示红色提示 + "切换手动录入" 按钮

### 表单区域

- 每个输入项使用可见 label（`--font-caption`，`--color-text-secondary`）
- 输入框高度 44px（满足触摸目标）
- 分类选择使用标签组样式（胶囊排列，选中高亮）
- 日期选择使用微信原生 `picker` 组件
- 过期时间自动计算结果展示在日期输入项下方，使用 `--color-info-bg` 底色卡片

### 保存按钮

- 全宽，`--color-primary` 背景，白色文字
- 高度 48px，`--radius-button` 圆角
- 保存中禁用 + 加载指示
- 保存成功后弹出激励反馈（弹窗动画使用 `--easing-bounce`）
