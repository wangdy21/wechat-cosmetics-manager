# 分类管理 (Category) — 页面设计覆盖

> 继承 `MASTER.md` 全部规范，以下为分类管理页特定的设计规则。

## 页面职责

查看预设分类、新增/编辑/删除自定义分类。

## 布局结构

```
┌──────────────────────────────┐
│  导航栏 (返回 + "分类管理")    │
├──────────────────────────────┤
│  预设分类区块                  │
│  (不可编辑，显示锁定图标)      │
├──────────────────────────────┤
│  自定义分类区块                │
│  (可编辑/删除)                │
├──────────────────────────────┤
│  添加分类按钮                  │
└──────────────────────────────┘
```

## 特定规则

### 预设分类

- 区块标题 "预设分类"（Caption）
- 每项显示：图标 + 分类名 + 右侧锁定图标（SVG lock，`--color-text-tertiary`）
- 不可点击编辑，使用 `opacity: 0.7` 暗示不可操作

### 自定义分类

- 区块标题 "自定义分类"（Caption）
- 每项显示：图标 + 分类名 + 右侧编辑/删除操作（滑动显示或直接展示）
- 删除使用 `--color-danger` 文字，需确认弹窗
- 空状态："还没有自定义分类"

### 添加按钮

- 底部固定或列表末尾
- 虚线边框样式 + 加号图标 + "添加分类"
- 点击弹出输入弹窗（名称 + 图标选择）
