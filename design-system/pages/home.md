# 首页 (Home) — 页面设计覆盖

> 继承 `MASTER.md` 全部规范，以下为首页特定的设计规则。

## 页面职责

概览仪表盘：即将过期产品警告、库存统计、最近添加。用户打开小程序的第一印象。

## 布局结构

```
┌──────────────────────────────┐
│  渐变装饰背景 + 几何图形       │
│  ┌──────────────────────────┐│
│  │  "我的美妆库" (H1)        ││
│  │  统计卡片行 (3列)         ││
│  └──────────────────────────┘│
├──────────────────────────────┤
│  即将过期警告区                │
│  (黄色/红色边框卡片列表)       │
├──────────────────────────────┤
│  最近添加                     │
│  (产品卡片列表)               │
├──────────────────────────────┤
│  底部 TabBar                  │
└──────────────────────────────┘
```

## 特定规则

### 顶部区域

- 背景使用多色渐变：`linear-gradient(135deg, #FFF0F3 0%, #EDE9FE 50%, #DCFCE7 100%)`
- 叠加半透明几何装饰（圆 + 三角 + 矩形）
- 统计卡片（在用/注意/安全率）使用大数字 + 渐变底色 + SVG图标
- 统计卡片间距 `--space-sm` (8px)

### 即将过期警告区

- 区块标题 "即将过期" 使用 H2
- 仅展示 `expirationDate` 距今 <= `advanceDays` 的产品
- 卡片使用 `--color-warning` 左边框 (4px solid)
- 已过期产品使用 `--color-danger` 左边框
- 卡片内展示：产品名、分类、剩余天数（大字突出）、进度条
- 无即将过期产品时显示激励文案："所有产品状态安全，继续保持！"

### 最近添加

- 最多展示最近 5 件，按 `createdAt` 倒序
- 使用紧凑卡片样式（图标 + 名称 + 分类 + 添加时间）
- 底部 "查看全部" 链接跳转库存页
