# 产品详情 (Detail) — 页面设计覆盖

> 继承 `MASTER.md` 全部规范，以下为产品详情页特定的设计规则。

## 页面职责

查看产品完整信息、标记用完/丢弃、删除。编辑功能为 P2。

## 布局结构

```
┌──────────────────────────────┐
│  导航栏 (返回 + "产品详情")    │
├──────────────────────────────┤
│  产品头部卡片                  │
│  (大图标 + 品牌名 + 产品名)    │
│  (分类标签 + 规格)             │
├──────────────────────────────┤
│  保质期状态卡片                │
│  (进度条 + 关键日期信息)       │
├──────────────────────────────┤
│  详细信息列表                  │
│  (生产日期/开封日期/保质期等)   │
├──────────────────────────────┤
│  操作按钮区                    │
│  (标记用完 / 标记丢弃 / 删除)  │
└──────────────────────────────┘
```

## 特定规则

### 产品头部

- 图标容器放大到 64x64px，渐变背景
- 品牌名（Caption）+ 产品名（H2）
- 分类标签使用 `--color-primary-bg` 底色 + `--color-primary` 文字
- 规格文字使用 `--color-text-secondary`

### 保质期状态卡片

- 突出展示剩余天数（`--font-stat` 28px）
- 使用语义色做整体色调（安全→绿、警告→黄、过期→红）
- 进度条放大到 8px 高
- 列出关键日期：生产日期、过期日期、开封日期（如有）

### 操作按钮区

- "标记用完" / "标记丢弃"：使用 `--color-text-secondary` 底色，不突出
- "删除"：使用 `--color-danger` 文字，视觉上与其他操作分离
- 删除前弹出确认弹窗
- 按钮高度 44px，圆角 `--radius-button`
