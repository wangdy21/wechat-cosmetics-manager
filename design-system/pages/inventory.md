# 库存页 (Inventory) — 页面设计覆盖

> 继承 `MASTER.md` 全部规范，以下为库存页特定的设计规则。

## 页面职责

全部产品清单：分类筛选、搜索、状态过滤。

## 布局结构

```
┌──────────────────────────────┐
│  搜索栏                       │
├──────────────────────────────┤
│  分类标签行 (横向滚动)         │
├──────────────────────────────┤
│  状态过滤标签                  │
├──────────────────────────────┤
│  产品卡片列表 (纵向滚动)       │
│  ┌────────────────────────┐  │
│  │ 图标 | 名称+分类 | 状态 │  │
│  │        进度条           │  │
│  └────────────────────────┘  │
│  ...更多卡片...               │
├──────────────────────────────┤
│  底部 TabBar                  │
└──────────────────────────────┘
```

## 特定规则

### 搜索栏

- 圆角 `--radius-input` (12px)，高度 40px
- 搜索图标使用 SVG，左侧内嵌
- 背景 `--color-surface`，`--shadow-card` 阴影
- 占位文字 "搜索产品名称或品牌..."

### 分类标签行

- 横向滚动，溢出不换行
- 胶囊样式，选中态 `--color-primary` 填充
- 每个标签显示对应分类的产品数量
- "全部" 标签始终在最左

### 状态过滤

- 使用小号胶囊，三个固定选项：在用 / 即将过期 / 已过期
- 使用对应语义色做背景（`--color-safe-bg` / `--color-warning-bg` / `--color-danger-bg`）
- 可多选或互斥（设计上建议互斥以降低复杂度）

### 产品卡片

- 使用完整产品卡片组件，点击跳转详情页
- 包含：图标容器（44x44，渐变底色）、产品名（H3）、分类+规格（Caption）、状态文字 + 进度条
- 卡片间距 `--space-md` (12px)
- 列表为空时显示空状态：SVG 插画 + "还没有产品，去添加第一件吧" + 跳转按钮

### 列表性能

- 使用微信小程序 `<recycle-view>` 或虚拟列表（超过 50 件产品时）
